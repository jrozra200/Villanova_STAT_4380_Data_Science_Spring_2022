Monte Carlo Simulation

What is a monte carlo simulation - write that out here.

Simple Example

get_rnorm <- function(nsamp) {
    tmp <- tibble(mean = mean(rnorm(nsamp)),
                  samp_size = nsamp)
    return(tmp)
}

test <- 1:100 %>% 
    map_dfr(~get_rnorm(10))

ggplot(test, aes(x = mean)) + 
    geom_bar()

Real World Example

Lets talk about being a successful mutual fund manager. A fund manager comes to you and says, Ive been successful for the past 10 years straight. Im a genius. Give me $10MM and Ill make you a fortune! What is your reaction?

The cool thing is that we can test this out and see for ourselves, using a Monte Carlo simulation, how skilled this person really is!

What if we assume that in any given year, irrespective of the year you had before, you have a 10% chance at being a successful mutual fund manager. Its hard and in other words, one years success is independent of the next years success. Lets also assume that there are a million mutual fund managers on day 1.

Is our person lucky? Or did they do something that required skill?

fund_success <- function(num_years = 10, success_pct = 0.5) {
    get_success <- tibble(
        year = 1:num_years,
        success = ifelse(runif(num_years) <= success_pct, TRUE, FALSE)
    )
    
    years_in_biz <- min(get_success$year[get_success$success == FALSE])
    
    if(is.infinite(years_in_biz)) {
        years_in_biz <- 10
    }
    
    return(years_in_biz)
}

num_mgr <- 10000

successful_fund_mgr <- 1:num_mgr %>% 
    map_dbl(~fund_success()) 

plot_dat <- tibble(mgr = 1:num_mgr,
                   years_successful = successful_fund_mgr)

ggplot(plot_dat, aes(x = successful_fund_mgr)) + 
    geom_bar()

Lets speed things up a bit with parallel processing.

future::plan(multisession)

num_mgr <- 100000

successful_fund_mgr <- 1:num_mgr %>% 
    future_map_dbl(~fund_success(), 
                   .progress = TRUE,
                   .seed = TRUE) 

plot_dat <- tibble(mgr = 1:num_mgr,
                   years_successful = successful_fund_mgr)

ggplot(plot_dat, aes(x = successful_fund_mgr)) +

This doesnt seem like much, right? You look at Wall Street stocked full of men (mainly men) dressed in high priced suits and driving expensive cars. Surely, this cant be the whole picture.

Well this is only one year. PLUS, we dont see the ones that wash out. They move away or work at the restaraunts that all the Wall St.guys go to. Lets see how this plays out over 10 years.

years <- 30
num_mgr <- 100000

successful_fund_mgr <- 1:(num_mgr * years) %>% 
    future_map_dbl(~fund_success(), 
                   .progress = TRUE,
                   .seed = TRUE) 

plot_dat <- tibble(year = rep(1:years, each = num_mgr),
                   mgr = rep(1:num_mgr, years),
                   years_successful = successful_fund_mgr) %>% 
    mutate(final_year = year + years_successful - 1,
           years_successful = as.character(years_successful))
